# Global-lotto
This repository contains a demonstration of a fake Global Lotto Sweepstakes website, recreated to help cybersecurity professionals, students, and the public understand how lottery scams operate. It is intended solely for education, awareness, and defensive security research. No real lottery, prize, or financial service is associated with this project.

Disclaimer: This project is provided exclusively for educational, awareness, and defensive cybersecurity purposes. It is not affiliated with any real lottery organization and must not be used to deceive, defraud, or impersonate any person or organization.
